# 🏗 천안IPC 통합현장 주간업무현황 대시보드

HDC현대산업개발 천안아이파크시티 통합현장 주간업무 관리 시스템

## 🌐 접속 주소
**https://i2002da.github.io/ipark-weekly-dashboard/**

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 📊 전체 현황 롤업 | 6개 현장(3~6단지·기반시설·DX TFT) 한 화면 요약 |
| ✏️ 공종별 마크다운 편집 | 토목·건축·전기·설비·조경·공무·RM 탭 분리 편집 |
| 👁 실시간 미리보기 | 분할/편집/미리보기 3가지 모드 |
| 📅 주요 일정 캘린더 | 대외·대내·DX·공정 일정 월별 관리 |
| 💾 GitHub Gist 동기화 | 현장별 Gist 분리 저장, ETag 충돌 방지 |
| 🔄 주차 이동 | 전/다음 주차 데이터 이동 |

## 🏢 현장별 공종 구분

| 현장 | 구분 | 작성 카테고리 |
|------|------|--------------|

| 기반시설 | 기반시설 | 토목 · 전기 · 통신 · 조경 · 공무 |
| DX TFT | DX | 현장 DX 적용 · 본사 유관팀 협의 · DX 교육·과제 |

## 📝 마크다운 상태 배지

작성 시 아래 키워드로 상태 배지가 자동 렌더링됩니다.

```
`완료`  →  초록 배지
`진행`  →  파란 배지
`예정`  →  노란 배지
`지적`  →  빨간 배지
```

## 🔐 접속 방법

1. 상단 GitHub PAT 입력창에 Personal Access Token 입력 (gist 권한 필요)
2. 최초 접속 시 현장별 Gist 자동 생성
3. 팀원 모두 동일한 PAT로 접속하면 실시간 공유

> PAT 없이 **데모 모드**로도 전체 기능 체험 가능합니다.

## 🛠 기술 스택

- **Frontend**: Vanilla JS + HTML/CSS (단일 파일 SPA)
- **스토리지**: GitHub Gist API (현장별 1개, 연도별 JSON 파일)
- **마크다운**: marked.js v9 (CDN)
- **배포**: GitHub Pages
